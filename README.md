### ABOUT

I specialize in creating accessible augmented reality experiences and computer vision applications that run entirely in the browser. My work focuses on practical solutions that don't require heavy installations—just open and interact.

Currently developing AR-based assistive technologies including a sign language tutor and cognitive training tools for Alzheimer's patients. When not coding, I direct short films to explore narrative-first thinking.

**Core Competencies:** Python • JavaScript • Java • React • TensorFlow • OpenCV • Mediapipe • Docker • WebXR

### APPROACH

My work philosophy centers on three principles:

**Accessible First** — Every project should run in a browser without installation barriers. WebAssembly, WebGL, and modern JavaScript frameworks enable sophisticated experiences that once required native apps.

**Production Ready** — Demos are interesting. Shipped products are valuable. I focus on complete, deployable solutions with real-world utility.

**Human-Centered** — Technology should adapt to users, not the other way around. This drives my work in assistive AR and accessible computer vision applications.
